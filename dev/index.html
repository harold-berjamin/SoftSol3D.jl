SoftSol3D.jl

A Julia package for the 3D incompressible quasi-linear viscoelasticity equations.

Version

Author

Installation

SoftSol3D.jl is an unregistered package, and is simply installed by downloading and extracting the files. Then, open Julia's REPL, type ] and run

pkg> add /home/JuliaUser/Downloads/SoftSol3D

where /home/JuliaUser/Downloads/SoftSol3D is the path to the SoftSol3D folder.

Citation

H. Berjamin (2023). "Computation of viscoelastic shear shock waves using finite volume schemes with artificial compressibility", Arxiv preprint 2310.04355. doi:10.48550/arXiv.2310.04355
